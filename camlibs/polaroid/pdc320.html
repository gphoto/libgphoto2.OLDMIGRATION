Polaroid PDC320 protocol description

Commands are prefixed with 4 E6.
name byte code command length reply length reply desc
Init 0x00 None. 1 byte, 0x05 Just returns 1 byte.
Id 0x01 None. 12 bytes, starting with 0x00 Returns 12 byte. Byte 1-9 are the string "XIRLINK", these are followed by a 0x02, 0x01 and 2 unknown additional bytes.
State 0x02 None. 22 bytes, starting with 0x02 Returns 22 bytes. First byte is 0x02.
End Init 0x0a no additional parameters 1 byte, 0x09 Just returns 1 byte.
